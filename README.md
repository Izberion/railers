# Railers

This is the official FoundryVTT system for Railers. 

## Key Features

- **Actor Sheets** - Complete character, NPC, train, and demon sheets with built-in automation for all calculated stats.
- **Drag and Drop Hex Map** - Quickly build the map as the crew explores with drag and drop hex tiles
- **Complete Compendiums** - Full compendiums featuring all gear, weapons, clothing, mutations, demons, and rollable tables.
- **Quick Weather and Temperature** - Quickly roll new weather and temperature as required
- **Custom Initiative Logic** - Fully integrated Railers initiative system that severs the token from the slot and reassigns it to the group or faction
- **Automated Reloading** - Easily track and manage ammunition and magazines, as well as reload empty firearms

## Installation

### Via FoundryVTT Browser
1. Open FoundryVTT and navigate to Game Systems
2. Click "Install System" and search for "Railers"
3. Click Install and then Activate

### Via Manifest URL
Use this manifest URL in FoundryVTT's system installer:
```
https://github.com/izberion/railers/releases/latest/download/system.json
```

### Requirements
- FoundryVTT v13 or later, verified on V14.
